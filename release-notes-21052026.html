Comparatif global — nouvelles fonctionnalités

Résumé des évolutions de l'outil pertinentes pour les animateurs réseau dans leur utilisation quotidienne.

Filtres par type d'événement

Nouveau Filtrer le tableau par type

Une barre de boutons colorés apparaît au-dessus du tableau comparatif. Cliquer sur un type — Apéro Quiz Apéro Concert Soirée Sport — affiche uniquement les métriques correspondant à ce type pour chaque établissement.

  • Les établissements sans publication de ce type affichent « Aucune pub de ce type »
  • Cliquer sur Tous réinitialise l'affichage complet
  • L'ordre alphabétique des établissements est conservé lors du filtrage

Nouvelles colonnes plateformes

Nouveau Répartition FB / IG / FB+IG

Trois nouvelles colonnes indiquent pour chaque établissement le nombre de publicités diffusées par plateforme sur la période sélectionnée.

  • FB — publicités sur Facebook uniquement (événements)
  • IG — publicités sur Instagram (objectif Trafic ou nom de campagne "Instagram")
  • FB+IG — publicités diffusées sur les deux plateformes simultanément

Indicateurs qualité

Nouveau Durée · Budget cible · Objectif

Trois colonnes de qualité affichent un pourcentage coloré pour chaque établissement sur la période :

  • Durée — part des pubs dont la durée atteint le minimum recommandé selon le type
  • Budget cible — part des pubs dont le budget total atteint ≥ 90 % du budget cible par type
  • Objectif — part des pubs avec un objectif recommandé (Interaction, Trafic ou Événement)

≥ 80 %  ≥ 50 %  < 50 %

Améliorations de navigation

Amélioré Sélection de période
  • L'année 2026 est présélectionnée par défaut à l'ouverture du comparatif
  • Le bouton Changer la période est maintenant vert et bien visible à côté du titre
  • Le bouton Recalculer (admin) est positionné à droite, séparé du bouton de navigation
Amélioré Lisibilité du tableau
  • Établissements triés par ordre alphabétique
  • Colonnes # et Établissement avec fond distinct pour faciliter la lecture sur les grands tableaux
  • Titres des colonnes FB / IG / FB+IG colorés pour correspondre aux cellules
  • Les données se chargent ligne par ligne au fur et à mesure — pas besoin d'attendre la fin pour consulter les premiers résultats
Action requise pour certains établissements
Les établissements dont les données comparatif ont été calculées avant cette mise à jour (Orléans Sud/Fleury, Montargis, Chambray et autres) n'ont pas encore les colonnes FB/IG/FB+IG ni les filtres par type. Cliquer sur Recalculer dans le topbar du comparatif pour rafraîchir leurs données.
